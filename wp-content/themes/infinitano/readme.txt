Description: Infinitano is a simple, minimalistic and responsive theme built with Twitter Bootstrap. The aim of this theme is to bring a pure Bootstrap experience in an extremely simple layout, forcing the user to focus on the content. It is thought to be used mainly in blogs, although future releases will bring support for more complex and complete websites. Features include custom logo (under Appearance > Customize), navigation menu with two-levels deep dropdown, ability to override default styles under Appearance > Editor. Supports the following 
languages: English.

-- Requirements --
==================

Infinitano requires WordPress 3.6+ installation to work properly.


-- Features --
=================

Theme has ability to adopt most popular screens easily, It is responsive by nature, Supports post thumbnails, Easy to read Font-Family, 
Flexible, Better and Clean Layout to focus on Content.


-- Installation --
==================

Just copy all the files from this direcory to your wordpress themes directory/ Infinitano. 
Then open admin panel of wordpress and activate this theme.


-- Configuration --
===================

Additionally this theme does not require any configuration. Just activate and enjoy.


-- Customization --
===================

To customize theme. Just go to Appearance>Themes and select the customization you want.
additionally theme supports - Header Logo changing, Custom Menus for Left side of logo and Right side of menu, Sidebar widgets. 
Background changing option.
